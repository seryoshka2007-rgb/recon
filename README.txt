========================================
   Local Recon Suite - Инструкция
========================================

Версия: 1.0
Дата: 2026


🔍 ЧТО ЭТО?
----------------------------------------
Local Recon Suite - это безопасный инструмент для сбора 
информации о системе Windows. Он собирает данные, которые 
могут пригодиться для аудита, диагностики или 
администрирования системы.

⚠️ ВАЖНО: Это НЕ взлом-инструмент. Скрипт НЕ крадёт пароли,
НЕ расшифровывает данные и НЕ оставляет следов в системе.
Это легальный инструмент для системного администратора.


📋 ЧТО СОБИРАЕТ?
----------------------------------------
Инструмент собирает следующую информацию:

1. СИСТЕМА (system)
   • Версия и название ОС
   • Время загрузки
   • Домен/рабочая группа
   • Производитель и модель ПК
   • Объём оперативной памяти

2. ПРОГРАММЫ (software)
   • Список всех установленных программ
   • Версии программ

3. ПОЛЬЗОВАТЕЛИ (users)
   • Список локальных пользователей
   • Список локальных групп

4. АВТОЗАГРУЗКА (autoruns)
   • Программы в автозагрузке (реестр)
   • Запланированные задачи

5. СЕТЕВЫЕ ПАПКИ (shares)
   • Общие сетевые папки на компьютере

6. СОХРАНЁННЫЕ УЧЁТНЫЕ ЗАПИСИ (creds_list)
   • Только ИМЕНА сохранённых записей
   • БЕЗ паролей и другой конфиденциальной информации


🚀 КАК ЗАПУСТИТЬ?
----------------------------------------
Самый простой способ:

1. Откройте флешку в Проводнике (Этот компьютер)
2. Найдите файл START_HERE.bat
3. Дважды кликните по нему (Запустить от имени администратора)
4. Дождитесь завершения (обычно 1-2 минуты)
5. Откройте папку reports
6. Найдите файл отчёта: local_recon_YYYYMMDD_HHMMSS.csv

Дополнительные способы запуска:

• autorun_launcher.bat  - альтернативный запускатор
• autorun.vbs          - скрытый запуск (без окон)
• local_recon_suite.exe --help - посмотреть все опции


📁 ФОРМАТ ОТЧЁТА
----------------------------------------
Отчёт сохраняется в формате CSV (Excel) в папке reports.

Имя файла: local_recon_ГГГГММДД_ЧЧММСС.csv
Пример:   local_recon_20260115_143022.csv

Структура CSV:
• category - категория (system, software, users и т.д.)
• item     - что найдено
• details  - подробности

Как открыть:
1. Дважды кликните по файлу
2. Откроется в Excel (или Notepad)
3. Можно фильтровать по колонке category

Пример фильтрации в Excel:
• Показать только установленные программы: 
  Фильтр category = software
• Показать только пользователей:
  Фильтр category = users


🔄 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ
----------------------------------------
Интерактивный режим (с вопросами):
  local_recon_suite.exe

Автоматический режим (без вопросов):
  local_recon_suite.exe --auto --format csv

Сохранить в JSON:
  local_recon_suite.exe --auto --format json

Сохранить в TXT:
  local_recon_suite.exe --auto --format txt

Выбрать только некоторые модули:
  local_recon_suite.exe --auto --modules=system,users,software

Полный список модулей:
  system,software,users,autoruns,shares,creds_list

Посмотреть справку:
  local_recon_suite.exe --help


🔒 ПРАВА ДОСТУПА
----------------------------------------
Для полного сбора данных рекомендуется запускать от имени 
администратора (ПКМ → Запуск от имени администратора).

Если запустить без прав администратора:
• Большинство данных будет собрано
• Некоторые задачи планировщика могут быть недоступны
• Доступ к чужим профилям пользователей может быть ограничен

Скрипт НЕ упадёт при отсутствии прав - просто пропустит 
недоступные данные и продолжит работу.

❓ ЧАСТЫЕ ВОПРОСЫ (FAQ)
----------------------------------------
В: Антивирус блокирует EXE-файл!
О: Это ложное срабатывание. Добавьте файл в исключения 
   антивируса или временно отключите защиту.

В: Не могу запустить от имени администратора!
О: Нажмите правой кнопкой на START_HERE.bat → 
   "Запуск от имени администратора"

В: Где найти отчёт?
О: В папке reports на флешке. Если папки нет - 
   создайте её вручную.

В: Сколько времени занимает сбор?
О: Обычно 1-2 минуты, зависит от мощности ПК и 
   количества установленных программ.

В: Могу ли я запустить несколько раз?
О: Да, каждый раз создаётся новый файл с уникальным 
   именем (по дате и времени).

В: Что делать, если скрипт завис?
О: Закройте окно и запустите заново. Если проблема 
   повторяется - перезагрузите компьютер.

В: Можно ли использовать на сервере?
О: Да, скрипт работает на Windows Server 2008/2012/2016/2019.

В: Какой формат отчёта лучше?
О: CSV - самый универсальный (открывается в Excel).
   JSON - для программистов.
   TXT - человекочитаемый, но без структуры.


📞 ТЕХНИЧЕСКАЯ ПОДДЕРЖКА
----------------------------------------
При возникновении проблем:

1. Проверьте, что запускаете от имени администратора
2. Убедитесь, что антивирус не блокирует программу
